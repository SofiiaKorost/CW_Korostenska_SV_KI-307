Data Recovery Software

# Опис
Цей репозиторій містить програму для відновлення втрачених файлів на комп'ютері. Програма написана на C++ і працює через консольний інтерфейс. Вона дозволяє сканувати диск у пошуках видалених файлів і намагатися їх відновити. 

# Основні функції:
1. Відображення доступних дисків для сканування
2. Аналіз структури файлової системи
3. Пошук видалених файлів
4. Фільтрація знайдених файлів за розширенням
5. Попередній перегляд знайдених файлів (назва, розмір, дата видалення, якщо можливо)
6. Відновлення вибраного файлу в задане місце
7. Створення журналу відновлених файлів
8. Мінімалістичний інтерфейс командного рядка

# Вимоги
- Компілятор C++ (наприклад, g++ або MSVC)
- ОС Windows або Linux

# Інструкція з використання
1. Скомпілюйте програму за допомогою `g++ main.cpp -o recovery`.
2. Запустіть `recovery.exe` або `./recovery`.
3. Дотримуйтесь вказівок у командному рядку.

**Примітка:** Дана програма не гарантує 100% відновлення файлів, оскільки деякі з них можуть бути перезаписані іншими даними.
